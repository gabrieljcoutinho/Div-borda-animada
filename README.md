# Div com borda animada


## Visão Geral

Este código implementa um elemento visual estilizado que exibe um **efeito de borda animada com gradiente rotativo**.  
O layout é simples e contém apenas uma caixa central (`.box`) com um texto interno.

Toda a parte dinâmica e visual é controlada exclusivamente por **CSS**, sem uso de JavaScript.

---

## Estrutura do HTML

O documento possui uma estrutura mínima:

- Um `body`
- Um `div` com classe `.box`
- Um parágrafo (`<p>`) dentro da caixa

A `.box` funciona como o container principal do efeito visual.

### Observação Importante – Erro de Marcação

Existe um pequeno erro no HTML:

Há uma **tag `</div>` extra** após o fechamento correto da `.box`.
